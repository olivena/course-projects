# Memory game

This program is programmed entirely with GitHub copilot, it is a memorygame working in browser.
It was firstly created as a lecture exercise in winter 2026, but continued as practice.

The game contains cards, game stats and scoreboard with 10 best scores. The gamecard pictures are emojis.

## To run the program
```
pip install flask
python app.py
```

## Details
- Python
- Flask library
- Functions in browser
- 64 cards
